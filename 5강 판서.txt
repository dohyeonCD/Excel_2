  - application 업데이트 끄기
  - 기존 데이터 지우기

@ 매크로 @
1. 일반 매크로 : 맨 땅에 헤딩
2. 자동 매크로 : 유저가 워크시트에서 메뉴를 통해 작업하는 과정을 엑셀이 자동으로 VBA 언어로 바꿔서 매크로 파일로 만들어 줌.
	        (처리 속도가 현저히 떨어짐. 자동매크로로 변환되지 않는 작업도 많음.)

자동 매크로는 딱 하나의 상황에서만 사용.
-> 알지 못하는 코드를 알고자 할 때 참고용으로-> 핵심 코드 찾기-> 구글링( vba 핵심코드).

<< 자동 매크로 돌리는 순서>>
1. 리본 개발 도구 메뉴-> '매크로 기록' (이제부터 내가 하는 거 잘 보고 바꿔줘~)
2. 수동 작업 (텍스트 나누기 : 해당 셀 잡고-> 리본 데이터 메뉴-> '텍스트 나누기')
3. 나 작업 끝났어~-> 리본 개발도구 메뉴-> '기록 중지'
4. VBA로 돌아가서 확인.
